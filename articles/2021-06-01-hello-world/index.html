What is this website built with?

This website is purely built with html and css. No trackers, no cookies stored. The website does use a bit of templating, but thats just used to make creating new articles easier, you can find all the source code here. The color scheme comes from Nord. If you want, you can get the data of any article by requesting the data.json. This will contain all the data you will need, including all the text, split into sections, then into paragraphs.
